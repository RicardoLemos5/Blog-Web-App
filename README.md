# Blog-Web-App
Blog Website that allows to Post, Edit and Delete posts.
Posts are listed in order of the most recent.

## How to start server
```npm install```

```nodemon index.js```
<br>
<br>
On your web browser navigate to [http://localhost:3000/](http://localhost:3000/)
